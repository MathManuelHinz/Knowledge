# Inhalt #EinfWth
Sei $A$ eine [Algebra](Einf.%20Wtheo/Definitions/Algebra.md).
Eine Abbildung $\mu:A\to[0,\infty]$ heißt Inhalt, falls
- $\mu(\emptyset)=0$;
- falls $B,C\in A$ mit $B\cap C=\emptyset$, dann gilt $\mu(A\cup B)=\mu(A)+\mu(B)$ #additivität
