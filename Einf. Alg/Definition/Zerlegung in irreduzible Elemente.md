# Zerlegung in [irreduzible](Einf.%20Alg/Definition/irreduzibel.md) Elemente #EinfAlg
Sei R ein [Integritätsbereich](Einf.%20Alg/Definition/Integrit%C3%A4tsbereich.md).
Eine ==Zerlegung== von $r\in R\setminus\{0\}$ ==in irreduzible Elemente== ist eine multiplikative Zerlegung
$$r=\epsilon\pi_1\cdot\dots\cdot\pi_k$$
mit einer [Einheit](Einf.%20Alg/Definition/Einheit.md) $\epsilon$ und irreduziblen Elementen $\pi_i$. Eine ==Zerlegung von r in Primelemente== ist eine Zerlegung mit einer Einheit und [Primelementen](Einf.%20Alg/Definition/Primelement.md) $\pi_i$. Wir sagen, dass R ein ==Integritätsbereich mit Zerlegung in irreduzible Elemente (bzw. Primelemte)== ist, wenn jedes $r\in R\setminus\{0\}$ eine solche Zerlegung gestattet. Wir sagen, dass R ein ==Integritätsbereich mit eindeutiger Zerlegung in irreduzible Elemente (bzw. Primelemte)== ist, wenn darüber hinaus diese Zerlegung im folgenden Sinn im wesentlichen eindeutig ist: Ist 
$$r=\tilde{\epsilon}\tilde{\pi}_1\cdot\dots\cdot\tilde{\pi}_\tilde{k}$$
eine weitere deratige Zerlegung, so gilt $k=\tilde{k}$ und es gibt eine Permutation $\sigma\in S_k$ mit $\pi_i\sim\hat{\pi}_{\sigma(i)}$, wobei $\sim$ die [multiplikative Äquivalenz](Einf.%20Alg/Definition/multiplikativ%20%C3%A4quivalent.md) ist.
## Bemerkung
- Weil auch $\epsilon\pi_1$ irreduzible ist, kann für $k>0$ immer $\epsilon=1$ angenommen werden.
- offenbar ist jeder integritätsbereich mit Zerlegung in Primelemente auch einer mit Zerlegung in irreduzible Elemente.
